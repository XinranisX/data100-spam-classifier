
# Data 100 Spam/Ham Email Classifier (Portfolio Refactor)

This repository refactors my UC Berkeley **Data 100** notebook project into a portfolio-friendly, reproducible codebase.

## Problem
Given an email text, predict whether it is **spam** (1) or **ham** (0).

## Data
- Input: `email` (text)
- Label: `spam` (0/1)
> Course datasets are not included in this repository.

## Current status
- Repository structure set up for an industry-style project (`src/` + `scripts/`)
- In progress: migrating feature engineering + model training from notebook into Python modules and CLI scripts

## Repository structure
- `src/`: reusable code (feature engineering, training, inference)
- `scripts/`: runnable entry points (train / predict)
- `notebooks/`: report-style notebook(s)
- `data/`, `models/`: local-only artifacts (ignored by git)
